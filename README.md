## Description

An RSS aggregator manages all of your RSS feeds in one convenient place.

## Application is based on:
- Axios
- Webpack
- Bootstrap

## Usage

1. installing deps inside repository
```
make install
```
2. Online development
```
make serve
```
3. Building bundle
```
make build
```
You can see the result in the directory dist/
